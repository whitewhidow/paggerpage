DECODE  (signal → numbers, all models)

ModelStationPagerActionSignal fingerprint
How to read this Every row decodes the same bits — the numbers alone can't tell you the model. Rows shown in red (✗) have a pager number or action value that hardware can't actually produce, so that model is ruled out. For the rest, match the "Signal fingerprint" against your Flipper read screen (its Frequency, Protocol/Bit, and Te values), then hit Use on the right row to load it into the generator.
Real pager capacity: T119 1–999 · TD157 1–998 · TD165 1–999 · TD174 1–10. Note: 433.92 MHz · Princeton 24bit · TE 271 fits both T119 and TD165 — pick your physical model.

About this file Generates one .sub that calls every pager in the station/pager ranges. Repeat = how many times each call is sent (default 10). For a single pager, set the from/to fields equal (the Use button does this for you).
